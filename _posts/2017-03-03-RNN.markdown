**ACTIVATION FUNCTIONS:**

**Above Identity** <br>

f(x) = x
<br> Derivative f'(x) = 1
<br>

<br>

<img src="https://upload.wikimedia.org/wikipedia/commons/thumb/f/fe/Activation_rectified_linear.svg/240px-Activation_rectified_linear.svg.png">

**Above ReLU**
f(x) = 0 for x < 0 <br>
f(x) = x for x >= 0 <br>
<br>
Derivative<br>
f'(x) = 0 for x < 0 <br>
= 1 for x >= 0 <br>
